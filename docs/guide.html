pyuring

Linux io_uring으로 파일 I/O를 하는 Python 바인딩이에요. 링 버퍼로 syscall을 줄이고, 배치·동시 I/O에서 특히 이득이 날 수 있어요.

커널 5.15+, Python 3.8+, 그리고 Linux에서 동작해요. (문서 본문은 영어 섹션이 많아요.) 
pip install pyuring
